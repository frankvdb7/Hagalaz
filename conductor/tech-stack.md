# Technology Stack

## Backend
- **Framework**: .NET 10 (C#)
- **Web**: ASP.NET Core
- **Orchestration**: .NET Aspire
- **Authentication**: OpenIddict (OAuth2 / OpenID Connect)
- **Resilience**: Polly
- **Messaging**: MassTransit with RabbitMQ

## Frontend
- **Framework**: Angular
- **Desktop Wrapper**: Electron

## Infrastructure & Data
- **Database**: MySQL
- **Caching**: Redis with FusionCache
- **Containerization**: Docker

## Communication
- **API Types**: REST, gRPC, WebSockets
- **Documentation**: OpenAPI / Swagger (Scalar)

## Observability
- **Standards**: OpenTelemetry (Metrics, Logs, Traces)
